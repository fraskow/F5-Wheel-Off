Normas de uso de la aplicación de Ruleta F5 Gaming:

  • Añadir tantos participantes como queramos para jugar.
  • Revisar el registro o lista de participantes.
  • Pulsar START para empezar a jugar.
  • El ganador será elegido de forma aleatoria.
  • El ganador será excluido del siguiente juego.
  • Botón RESET para limpiar o resetear la lista de jugadores.
